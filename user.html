Privacy Policy

Effective Date: June 3, 2026. User data is protected and used only for educational services provided by Ed Achievers.

1. Information Collection

We collect essential data to authenticate students, register them into preparatory courses, process payments safely, and track educational progress (like quizzes and notes access).

2. User Account Information

When you register using Firebase Auth, we save credentials like your name, email address, password hash, and referee patterns to establish stable access states. This data is private and never traded.

3. Payment Information

Payment information is processed using secured third-party services like PayU Checkout. We never cache or store raw UPI, CVV, or net-banking attributes locally, relying fully on official redirect API structures.

4. Data Storage

Student registration records, purchased packages lists, test marks, and doubt posts are synchronized securely on Firebase Firestore/Realtime Database systems or cached in LocalStorage configuration directories.

5. Security Measures

We implement active database access rules, cryptographic signature signing for checkouts, and SSL connection layers to safeguard student profiles from unauthorized intrusion vectors.

6. Cookies Usage

We use essential functional browser cookies and localStorage parameters to keep your profile active during sessions, saving you from repeating sign-ins continuously.

7. Third Party Services

To improve your learning experience, we use trusted external tools like YouTube SDK, Firebase services, and PayU APIs. These third-party services adhere to independent security constraints.

8. User Rights

Every student retains the right to download their progress logs, modify account display names, register for doubts resolution, and revoke sessions. For administrative data updates, reach out to our helpdesk.

9. Contact Information

For inquiries regarding privacy, cookie settings, or to request personal profile removal, email support@edachievers.in.
